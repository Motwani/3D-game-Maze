3D GAME by Aditya Motwani
201401206


Implemented Camera Rotate and Camera View.
Block Height is Random
Basic Platform is Generated and Player Movement.
Make File Included.
Command :

g++ -o game maze.cpp glad.c -lGL -lglfw -L/usr/local/lib -ldl

Run the file "game"

Controls:

Left Key : Rotate Camera Left
Right Key : Rotate Camera Right
Up Key : Move Camera Up
Down Key : Move Camera Down
T Key : Toggle between Top View and Tower View

A,S,W,D Key - Move Player
